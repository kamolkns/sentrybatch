Setup

Open v1.html in any modern browser. Nothing to install. API keys are kept only for the current browser session and are cleared when the session ends. They are sent only to VirusTotal / AbuseIPDB, or to the CORS proxy you explicitly configure.

Browser CORS note: VirusTotal and AbuseIPDB do not send CORS headers, so browsers block direct fetch() calls to them from a page like this one — you'll see "Blocked by browser (CORS)" or "Failed to fetch" even with a valid key. This isn't a bug in the key or the request; it's the browser refusing to hand back the response. Fix it by setting a CORS proxy prefix in Settings, for example:
  • A public proxy for quick testing, e.g. https://corsproxy.io/?url= — simple, but your API key passes through that third party, so only use a throwaway/low-privilege key this way.
  • A tiny self-hosted proxy (safest) — a one-file Cloudflare Worker, Vercel edge function, or Node/Express endpoint that forwards the request server-side and adds permissive CORS headers back to the browser. This keeps your key off any third party.
Geolocation (ipapi.co) and DNS resolution (dns.google) are CORS-friendly already and work without a proxy.

Bulk input format

This tool checks IP addresses and domains only — file hashes, URLs, or other indicator types (e.g. a 64-character SHA-256 string) aren't recognized and are skipped with a warning in the log. Use VirusTotal directly for hash/file lookups.

Bulk input

  • Paste IPs one per line, or comma/space separated.
  • CIDR blocks (e.g. 192.168.1.0/28) and hyphen ranges (e.g. 10.0.0.1-10.0.0.5) are auto-expanded.
  • Domains are resolved to an IP via Google's DNS-over-HTTPS API before checking.
  • Duplicates are removed automatically. Lists over 1000 unique entries are blocked from processing.

Scoring

Threat Score is calculated from VirusTotal malicious/suspicious engine detections, with an optional AbuseIPDB contribution. A result with zero VT detections is shown as No detections reported, not “safe”; it is not a guarantee that an IP is harmless. If VirusTotal cannot be reached, the score is shown as unavailable. Results are cached for 1 hour per IP.

Rate limits

VirusTotal's public tier allows 4 requests/minute — the queue throttles automatically. Change the tier in Settings if you have a premium key.

All processing happens locally in your browser. No analytics, no tracking, no server component.
