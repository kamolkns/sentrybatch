How this works

Setup

Open index.html in any modern browser. Nothing to install. API keys are kept only for the current browser session and are cleared when the session ends. They are sent only to VirusTotal / AbuseIPDB API endpoints via a built-in CORS proxy (https://corsproxy.io/) so no manual proxy configuration is needed.

Bulk input format

This tool checks IP addresses and domains only — file hashes, URLs, or other indicator types (e.g. a 64-character SHA-256 string) aren't recognized and are skipped with a warning in the log. Use VirusTotal directly for hash/file lookups.

Bulk input

  • Paste IPs one per line, or comma/space separated.
  • CIDR blocks (e.g. 192.168.1.0/28) and hyphen ranges (e.g. 10.0.0.1-10.0.0.5) are auto-expanded.
  • Domains are resolved to an IP via Google's DNS-over-HTTPS API before checking.
  • Duplicates are removed automatically. Lists over 1000 unique entries are blocked from processing.

Scoring

Threat Score is calculated from VirusTotal malicious/suspicious engine detections, with an optional AbuseIPDB contribution. A result with zero VT detections is shown as No detections reported, not “safe”; it is not a guarantee that an IP is harmless. If VirusTotal cannot be reached, the score is shown as unavailable. Results are cached for 1 hour per IP.

Rate limits

VirusTotal's public tier allows 4 requests/minute — the queue throttles automatically. Change the tier in Settings if you have a premium key.

All processing happens locally in your browser. No analytics, no tracking, no server component.
